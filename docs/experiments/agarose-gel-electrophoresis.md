# Agarose Gel Electrophoresis

## Purpose

This is the recommended first experiment for Phase 1 because it has clear visual steps, strong training value, and realistic mistakes that can be simulated.

## Required Training

- Lab safety basics
- PPE usage
- Micropipette handling
- Gel electrophoresis equipment orientation

## Required Objects

- Agarose
- Buffer
- DNA sample
- Loading dye
- DNA ladder
- Casting tray
- Comb
- Gel tank
- Power supply
- Micropipette
- Waste container
- Gloves
- Lab coat
- Safety goggles

## SOP Flow

1. Wear PPE.
2. Clean the workbench.
3. Collect all required materials.
4. Measure agarose quantity.
5. Prepare buffer.
6. Heat agarose solution safely.
7. Pour gel into casting tray.
8. Insert comb.
9. Wait for gel to solidify.
10. Place gel in tank.
11. Add running buffer.
12. Mix sample with loading dye.
13. Load ladder and samples into wells.
14. Connect electrodes correctly.
15. Set voltage and time.
16. Run gel.
17. Observe result.
18. Dispose waste properly.
19. Generate report.

## Mistakes To Track

- Entering lab without PPE
- Skipping bench cleaning
- Wrong agarose concentration
- Wrong buffer volume
- Touching sterile materials with contaminated gloves
- Damaging gel while removing comb
- Loading wrong well
- Reversing electrodes
- Setting voltage too high
- Incorrect waste disposal

## Result Logic

Final result quality should depend on:

- Correct sequence
- Measurement accuracy
- Sterility
- Instrument settings
- Waste handling
- Time management

Possible results:

- Excellent
- Acceptable
- Contaminated
- Failed
